# challenge11-ExpressNotes

## Description

This repository holds the code for a webapp deployed on heroku where you can view, create and delete notes. It's creation was a study in creating post, get, and delete routes.
It functions with Nodejs, with most of the work being in understanding how to effectively create models and routes that function. The app allows for users to manage
notes that are saved on the heroku server, by being able to view, create, and delete notes.
